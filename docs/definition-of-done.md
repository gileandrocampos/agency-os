# Definition of Done

Uma feature só pode ser considerada concluída quando atender TODOS os critérios abaixo:

- [ ] Implementação finalizada
- [ ] Responsabilidade única respeitada
- [ ] Código revisado
- [ ] Testes unitários criados
- [ ] Todos os testes passando
- [ ] Cobertura de testes ≥ 90%
- [ ] Safe-crawl executado com sucesso
- [ ] Logs implementados
- [ ] Tratamento de erros implementado
- [ ] Documentação atualizada
- [ ] CHANGELOG atualizado
- [ ] ROADMAP atualizado (quando necessário)
- [ ] Branch pronta para merge
